# GoD_frontend

The Game of Drones frontend was developed using Angular 11.2.3 with Node.js 12.21.0

# Deployment Process 

1. On you have the Backend installed and running edit the $/assets/God_config.json to matched the exposed operations
   change "apiBaseUrl" : "http://localhost:5000/api/Game" to the actual URL of the running api.

2. Start the web site....
